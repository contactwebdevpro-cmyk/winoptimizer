Questions fréquentes

Est-ce que ce script est sûr à utiliser ?

Oui. Le code est entièrement open source et disponible sur GitHub. Il n'installe aucun programme, ne modifie pas le registre de manière irréversible et n'effectue aucune action réseau autre que son propre téléchargement. Vous pouvez lire chaque ligne avant de l'exécuter.

Sur quelles versions fonctionne-t-il ?

Le script est compatible avec les éditions courantes 64 bits. Il détecte automatiquement votre version et adapte les opérations en conséquence. Un compte administrateur est requis pour appliquer l'ensemble des optimisations.

Puis-je annuler les modifications ?

Un point de restauration système est créé automatiquement avant toute modification. En cas de problème, vous pouvez revenir à l'état précédent depuis les options de récupération de votre système en quelques clics.

Faut-il relancer le script après une mise à jour ?

Certaines mises à jour majeures peuvent réactiver des paramètres modifiés. Il est recommandé de relancer le script après une mise à jour importante pour s'assurer que toutes les optimisations restent appliquées.
